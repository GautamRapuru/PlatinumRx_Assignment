PlatinumRx

Overview
This project is a data analyst assignment where I worked on solving practical problems using SQL, Excel, and Python. The focus was on understanding the data and applying the right techniques to extract useful insights.

Tools Used
•	SQL (SQLite)
•	Microsoft Excel 
•	Python


SQL – Hotel System
Created tables and analyzed booking and billing data.
Tasks included:
•	Finding the latest booking room for each user
•	Calculating monthly billing
•	Filtering high-value bills
•	Identifying most and least ordered items
•	Finding the second highest bill per month


SQL – Clinic System
Created Tables and Worked with clinic sales and expense data.
Tasks included:
•	Revenue by sales channel
•	Identifying top customers
•	Monthly profit and loss analysis
•	Finding most and least profitable clinics

Spreadsheet Analysis
Used Excel to combine and analyze ticket and feedback data:
•	Applied XLOOKUP to fetch related data
•	Checked tickets closed on the same day and same hour
•	Used functions like COUNTIFS, INT, and HOUR



Python
•	Converted minutes into hours and minutes
•	Removed duplicate characters from a string using loops


How to Run
•	Execute SQL files in any SQLite environment
•	Open the Excel file to view formulas and analysis
•	Run Python scripts using Python 3.9
